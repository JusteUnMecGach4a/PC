# 🔒 Projet PC - Alerte Sécurité

Ce projet est une page web interactive conçue pour rappeler aux utilisateurs de verrouiller leur poste de travail.

## 🚀 Aperçu

La page affiche un message d'alerte dynamique avec des animations visuelles percutantes pour attirer l'attention sur les raccourcis de verrouillage :
- **Windows** : `Win + L`
- **Linux/Ubuntu** : `Ctrl + Alt + L`

## 🌐 Déploiement

Le site est automatiquement déployé via GitHub Pages.
Vous pouvez le consulter ici : [https://justeunmecgach4a.github.io/PC/](https://justeunmecgach4a.github.io/PC/)

## 🛠️ Technologies utilisées

- HTML5
- CSS3 (Animations, Flexbox, Glassmorphism)
